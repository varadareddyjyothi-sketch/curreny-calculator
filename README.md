# CurrencyX

## Overview

CurrencyX is a modern, responsive currency converter website built with HTML, CSS, and JavaScript. It lets users convert between major and global currencies using live exchange rates.

## Features

- Real-time conversion using a free exchange-rate API
- Searchable currency dropdowns
- Swap currencies button
- Popular conversion options
- Recent conversions saved in local storage
- Dark / light theme toggle
- Responsive glassmorphism UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Local Storage

## Installation

1. Clone or download the project.
2. Open `index.html` in your browser, or run a local server.
3. For a local server, use:

   ```bash
   python -m http.server
   ```

## API Setup

This project uses the free Frankfurter API for live exchange rates:

- Base URL: `https://api.frankfurter.app`
